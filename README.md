# PitoySenpai - Anime Streaming Website

A simple anime streaming website with user authentication and movie catalog.

## Features
- User Login & Signup
- Browse 30+ Anime Movies
- User Account Management
- Admin Panel for User Management
- Time Tracking for Browsing

## Pages
- `index.html` - Home page with movie catalog
- `login.html` - User login page
- `signup.html` - User registration page
- `account.html` - User account page
- `admin.html` - Admin panel
- `edit.html` - Edit user information

## Technologies
- HTML5
- CSS3
- PHP (for backend functionality)
- MySQL (for database)

## Setup
1. Clone the repository
2. Set up a local server (XAMPP, WAMP, etc.)
3. Import the database
4. Configure `db.php` with your database credentials
5. Access via `localhost`

## Note
The HTML files are static versions for GitHub Pages preview. Full functionality requires PHP backend.
